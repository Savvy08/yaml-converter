# ClashConfigManager
Чтобы воспользоваться конвертором, скачиваем программу [Здесь](https://github.com/Savvy08/yaml-converter/releases). 
Как работает? Программа удаляет ненужные конфигурации и преобразует их в localhost, который доступен по порту 8080.

-----

#### Для чего создана:

Все конверторы в yaml -> конвертируют в локальный файл.yaml, 
либо к подписке добавляют CN домены и правила, меняя названия протоколов, что доставляет дискоморфт при использовании.

-----

localhost:8080 - что это за адрес и для чего он нужен?
Программа запускает сервер localhost, чтобы данные о подписках могли передаваться в программы из семейства clash-meta.
Не занимает много места.

**В данный момент поддерживает ссылки:**
- _https://домен/iduaUI..../c051f231ss828..../clashmeta/?asn=text#Clash-meta%TEXT_
- _https://sub.yorun.me/api?type=clash&filename=123&url=https://домен_

-----

#### Где взять конвертированную ссылку:
1. Переходите на этот [сайт](https://sub.yorun.me/)
2. Вставляете ссылку подписки
3. Далее переходите в праграмму


#### Как пользоваться Calsh Config Manager:
1. Распаковать/Установить программу
2. Запустить от (имени администратора)
3. Вставить ссылку подписки -> после конвертирования, создаст файл config.yaml
4. Скопировать localhost
5. Зайти в Clash Verge или его аналоги
6. Включить ядро "Альфа или Смарт" в настройках
7. Вставить ссылку на localhost
8. По усмотрению поставить в автозапуск (в программе отображается подписка)
